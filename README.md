# JavaScript Practice Workspace

This repository is my personal step-by-step notebook for mastering JavaScript.

## Project Structure
* `pract.html` - The main browser window used to test all experiments.
* `01_data_types.js` - Practice code for JavaScript basic data types and conversions.
* `02_control_flow.js` - Code containing conditional statements (ATM, Traffic Light simulators).
* `03_functions.js` - Code practicing parameters, arguments, and return values.
* `04_arrays.js` - Practice code for arrays, structural indexing, and list manipulation methods.
* `05_objects.js` - Practice code for JavaScript objects, dot notation mutation, and property validation.
* `06_loops.js` - Practice code for execution iterations, loop termination handling, and data accumulation engines.
* `07_methods.js` - Practice code for encapsulated object behavior, state mutation, and internal scoping using 'this'.

### 🏆 Independent Checkpoints
* `bible_journey.js` - **Milestone 1 (Core Logic Engine):** An interactive Old Testament Journey to Canaan simulator that runs as a finite state machine using object state mutations, loop sequencing, and strict resource tracking conditional logic.
* `armor_tracker.js` - **Milestone 1 (Alternative Variant):** An Armor of God devotional application tracking spiritual health (`faithPoints`) against array collections of daily mental and physical trials.

## How to Run
To switch between topics or checkpoints, open `pract.html` and update the `<script src="...">` tag at the bottom to target the file you want to review, then open the HTML file in your browser and check the developer console (`F12`).
