# JavaScript Practice Workspace

This repository is my personal step-by-step notebook for mastering JavaScript.

## Project Structure
* `pract.html` - The main browser window used to test all experiments.
* `01_data_types.js` - Practice code for JavaScript basic data types and conversions.
* `02_control_flow.js` - Code containing conditional statements (ATM, Traffic Light simulators).

## How to Run
To switch between topics, open `pract.html` and update the `<script src="...">` tag at the bottom to target the file you want to review, then open the HTML file in your browser.
